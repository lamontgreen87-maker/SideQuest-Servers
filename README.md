# SideQuest Server Registry

**Community-maintained list of public SideQuest servers.**

SideQuest is a self-hostable AI-powered D&D dungeon crawler. This registry allows anyone to discover and connect to community-run servers.

## Using This Registry

The web client automatically fetches this server list from:
```
https://raw.githubusercontent.com/lamontgreen87-maker/SideQuest-Servers/main/servers.json
```

## Running Your Own Server

Want to host your own SideQuest server? Check out the [hosting guide](https://github.com/lamontgreen87-maker/SideQuest/blob/main/HOSTING.md).

## Adding Your Server
